3	程序设计详细描述
《货品基本信息维护》
	该模块用于查询货品信息并列表显示。
	功能简述：分页列出所有货品，具有权限的操作员可以浏览、新增、修改、删除。
3.1	货品基本信息维护
模块编号	M001		源程序文件	ProductList.html
功能	货品浏览
输入参数	货品信息表（T_Product）页面上所有对应栏位值（参表结构）
要访问的表	货品信息表（T_Product）
模块时序图	 
图：时序图
输入界面	首页为浏览输入页，如下图：
 图：浏览输入页面

输出界面	下图为货品浏览的输出页
 
图：浏览页面

主要控件说明：
	点击“Create New”链接，跳转至新增页面；
	点击“Edit”链接，跳转至修改页面；
	点击“Details”链接，跳转至详细页面；
	点击“Delete”链接，跳转至删除页面。
出错处理	统一跳转至error.html页面
