## 判题状态数据流

```mermaid
stateDiagram-v2
    [*] --> Pending: 用户提交代码
    Pending --> Judging: @Async JudgeExecutor
    Judging --> Compiling: Go-Judge compile()
    Compiling --> CE: 编译失败
    Compiling --> Running: 编译成功
    Running --> AC: 全部用例通过
    Running --> WA: 输出不匹配
    Running --> TLE: 运行超时
    Running --> MLE: 内存超限
    Running --> RE: 运行时错误
    Running --> OLE: 输出超限
    Running --> PA: 部分用例通过
    Running --> SE: 系统错误
    AC --> [*]
    CE --> [*]
    WA --> [*]
    TLE --> [*]
    MLE --> [*]
    RE --> [*]
    OLE --> [*]
    PA --> [*]
    SE --> [*]
```
